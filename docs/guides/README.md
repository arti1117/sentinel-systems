# 📚 자율주행 지식 베이스 마스터 인덱스 (Sentinel Systems 커리큘럼)

Sentinel Systems가 제안하는 **자율주행 아키텍처 및 관제 인프라의 단계별 가이드**입니다.
이 커리큘럼은 대상 청중(청자와 비즈니스 역할)에 따라 깊이가 다르게 설계되어 있습니다. 자신의 수준과 역할에 맞는 문서를 가장 먼저 확인하십시오.

---

## 🗺️ 문서 네비게이션 구조

| 단계 | 대상 독자군 | 핵심 학습 목표 |
|---|---|---|
| **[101](./자율주행-101-생초보-특강.md)** | 비전문가, 일반 관리자 | 자율주행 3단계 과정의 직관적 파악 및 관제 시스템의 사용자 시나리오(UX) 파악 |
| **[201](./자율주행-201-초중급-특강.md)** | 초/중급 개발자, PM | ROS 2 통신 미들웨어의 흐름 이해와 가상-현실 간 간극(Sim-to-Real Gap)의 이해 |
| **[301](./자율주행-301-엔지니어-딥다이브.md)** | 시니어 로보틱스 엔지니어 | 방향성 비순환 그래프(DAG) 레이턴시 통제와 확장 칼만 필터(EKF)의 수학적 적용 체계 |
| **[401](./자율주행-401-시니어-아키텍트.md)** | 시스템 아키텍트 | OS 레벨 결정론(Determinism) 보장 및 멀티스레드 교착 상태(Deadlock) 회피 설계 |
| **[501](./자율주행-501-최고전문가-CTO.md)** | 경영진, CTO, 투자자 | 법적 책임 방어(Liability Transfer)와 형식 승인을 위한 거버넌스(Governance) 전략 |
| **[601](./자율주행-601-학계-개방형난제.md)** | 학계 연구자, 박사급 | 신경형 센서, 분포 외(OOD) 추론의 모순 등 자율주행 기반의 근본 한계 돌파 이론 |

> **읽기 권장 순서**
